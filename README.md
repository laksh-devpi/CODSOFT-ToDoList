# 📝 To-Do List Application

A simple command-line To-Do List application built using Python.

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- User-friendly menu
- Simple and easy to use

## Technologies Used

- Python 3

## How to Run

1. Clone the repository:
   git clone https://github.com/laksh-devpi/CODSOFT-ToDoList.git

2. Open the project folder.

3. Run:

   python todo.py
